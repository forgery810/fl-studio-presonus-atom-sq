# Presonus Atom SQ MIDI Script for FL Studio 

A python MIDI-script to increase the functionality of the Presonus Atom SQ with FL Studio. This script adds a step-sequencer, random step and random note generator, mixer, channel, and plugin control, etc.
 
# Version 1.0 - Complete Revision

This updated version completely restructures the code. All buttons and knobs are now able to assigned functionality by users. The more experimental aspects of the previous version have been eliminated. More functions have been added with a focus on live performance through the Channel Rack.
